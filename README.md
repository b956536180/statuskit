# Introduction 介绍

StatusKit 是一个基于 Hexo 的静态 Status Page 生成器。  
StatusKit is a static state page generator, based on Hexo.

- List your service components
- Report incidents
- Markdown support for incident messages
- RSS supported

# Install 安装

```bash

cd statuskit
npm install
```
